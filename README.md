# Tripadvisor - Travel Guide Web App

A simple and interactive travel guide web application built with HTML, CSS, and JavaScript. This project showcases popular destinations, user reviews, and travel tips, providing users with a mini digital travel companion.

## Features

- Browse through a list of top travel destinations
- View detailed information about each location
- User reviews and ratings
- Responsive design for mobile and desktop
- Interactive elements with JavaScript
